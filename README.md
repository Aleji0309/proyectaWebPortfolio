## 📋 Project Description

This web portfolio showcases a collection of projects and skills developed during the program. It’s designed to be clean, responsive, and accessible — highlighting both aesthetics and functionality.  
It serves as both a personal website and a space to showcase development challenges and projects.

## 📌 Project Goals

- Apply and consolidate front-end skills (HTML, CSS, JS)  
- Create a personal, customizable and professional web portfolio  
- Practice clean structuring and layout techniques  
- Publish the project using Netlify

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Aleji0309/proyectaWebPortfolio.git
````

2. Navigate to the project directory:

```bash
cd proyectaWebPortfolio
```

3. Open the `index.html` file in your browser, or use a local server like **Live Server** (in VS Code).
